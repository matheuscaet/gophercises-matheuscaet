
# Gophercises

Welcome to my Gophercises solutions repository! This project contains implementations of coding exercises designed to improve Go programming skills.

## Overview

Gophercises is a collection of practical Go exercises that cover various programming concepts and techniques, from basic fundamentals to more advanced patterns.

## Exercises Included

- Exercise 1: Quiz Game
- Exercise 2: URL Shortener
- Exercise 3: Choose Your Own Adventure
- Exercise 4: Link Parser
- Exercise 5: Sitemap Builder
- Exercise 6: Hacker Rank CLI
- Exercise 7: Temp Mail Handler
- Exercise 8: Phone Number Formatter
- Exercise 9: Blackjack Game
- Exercise 10: Decode JSON
- Exercise 11: Phonebook API
- Exercise 12: REST API Crud

## Getting Started

1. Clone the repository
2. Navigate to individual exercise directories
3. Read each exercise's README for specific instructions
4. Run the solutions with `go run` or `go build`

## Prerequisites

- Go 1.16 or higher
- Basic understanding of Go syntax
